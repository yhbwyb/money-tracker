# 记账本安装教程

## 你需要准备

- 一台 iPhone
- 一个 GitHub 账号（没有的话去 github.com 免费注册，2 分钟）
- 翻墙工具（安装时需要，日常用不需要）

---

## 第一步：复制代码仓库

1. 电脑浏览器打开 https://github.com/yhbwyb/money-tracker
2. 点右上角 **Fork** 按钮
3. 页面刷新后，代码就复制到你自己的账号下了

---

## 第二步：开启网页托管

1. 在你刚 Fork 的仓库页面，点顶部 **Settings**
2. 左侧菜单点 **Pages**
3. Branch 下拉选 **gh-pages**，点 **Save**
4. 等 2 分钟

> 你的网址是：`https://你的用户名.github.io/money-tracker/`

---

## 第三步：装到手机上

1. iPhone **打开翻墙**
2. 用 **Safari** 打开上一步的网址
3. 确认页面能正常显示
4. 点底部中间 **分享按钮**（⬆ 方框箭头图标）
5. 往下滑，点 **添加到主屏幕**
6. 名称确认是「记账本」，点右上角 **添加**
7. 桌面出现图标，**关掉翻墙**
8. 点开桌面图标，看看能不能正常打开

---

## 以后怎么用

- 日常记账：直接点桌面图标，**不需要翻墙，不需要网络**
- 数据备份：进入「印鉴」→ 导出留底 → JSON 文件存手机
- 版本更新：开翻墙 → 点开一次 → 自动更新 → 关翻墙

---

## 常见问题

**打开是白屏？** → 开翻墙重新打开一次，加载完就好了

**数据会丢吗？** → 不删图标就不会。建议每月导出留底一份

**支持安卓吗？** → 支持，Chrome 打开网址 → 菜单 → 添加到主屏幕
